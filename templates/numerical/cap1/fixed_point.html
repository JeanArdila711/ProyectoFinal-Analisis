Descripción

El método de punto fijo es una técnica para resolver ecuaciones no lineales que consiste en reformular la ecuación original f(x) = 0 en la forma x = g(x), donde g(x) es una función auxiliar que se elige para que sus iteraciones sucesivas converjan a una raíz de f(x).

A partir de un valor inicial x₀, se evalúa iterativamente g(x), generando una secuencia: x₁ = g(x₀), x₂ = g(x₁), x₃ = g(x₂), ... Los valores obtenidos tienden a aproximarse cada vez más a la raíz si la función g(x) cumple ciertas condiciones de convergencia.

Condiciones de Convergencia:

Para que el método converja, g(x) debe cumplir:

  1. |g'(x)| < 1 en un intervalo alrededor de la raíz
  2. g(x) debe ser continua
  3. El punto fijo p cumple: p = g(p)
Algoritmo:
  1. Reformular f(x) = 0 como x = g(x)
  2. Comenzar con un valor inicial x₀
  3. Calcular x₁ = g(x₀)
  4. Calcular x₂ = g(x₁)
  5. Repetir hasta que |xn+1 - xn| < tolerancia

Cómo Construir g(x)

Existen varias formas de despejar x = g(x) desde f(x) = 0:

Ejemplo 1: x² - 4 = 0
  • Opción A: x = √4 → g(x) = 4/x
  • Opción B: x = x² - 4 + x → g(x) = (x² + 4)/(2*x)
Ejemplo 2: x - cos(x) = 0
  • Opción A: x = cos(x) → g(x) = cos(x)
  • Opción B: x = x - (x - cos(x)) → g(x) = cos(x)
Importante: No todas las formas de g(x) convergen. Verifica que |g'(x)| < 1 cerca de la raíz.

Ejemplos

Ejemplo 1: Raíz cuadrada
  • f(x): x**2 - 4
  • g(x): 4/x
  • x₀: 1
  • Tolerancia: 5e-5
  • Raíz esperada: 2
Ejemplo 2: Ecuación trigonométrica
  • f(x): x - math.cos(x)
  • g(x): math.cos(x)
  • x₀: 0.5
  • Tolerancia: 5e-5
  • Raíz esperada: ≈ 0.7391
Ejemplo 3: Ecuación exponencial
  • f(x): math.exp(x) - x - 2
  • g(x): math.log(x + 2)
  • x₀: 1
  • Tolerancia: 5e-5
