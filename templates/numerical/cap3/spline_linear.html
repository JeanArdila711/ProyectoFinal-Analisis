Descripción

Un Spline Lineal es la forma más simple de interpolación por tramos. Conecta cada par de puntos consecutivos con una línea recta.

Características:

  • Muy simple - Solo requiere conectar puntos con líneas
  • Computacionalmente rápido - Cálculos mínimos
  • Funciona siempre - No tiene problemas de convergencia
  • No suave - Tiene "esquinas" en los puntos
  • Primera derivada discontinua - Las pendientes pueden cambiar bruscamente

Fórmula del Spline Lineal

Para cada intervalo [xi, xi+1]:

Si(x) = mi · (x - xi) + yi

Donde la pendiente mi es:

mi = (yi+1 - yi) / (xi+1 - xi)

Ventajas matemáticas:

  • Primera derivada (pendiente) es constante en cada tramo
  • No requiere resolver ecuaciones lineales
  • Garantizado pasar exactamente por los puntos

Ejemplos

Ejemplo 1: Función Simple
  • X: 0 1 2 3
  • Y: 0 1 4 2
  • Resultado: 3 segmentos de recta conectando los puntos
Ejemplo 2: Función Escalonada
  • X: 0 1 2 3 4
  • Y: 1 1 2 3 3
  • Resultado: Líneas con diferentes pendientes
Ejemplo 3: Triángulo
  • X: 0 1 2
  • Y: 0 2 0
  • Resultado: Forma triangular
