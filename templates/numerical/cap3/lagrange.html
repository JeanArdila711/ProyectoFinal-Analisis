Descripción

La Interpolación de Lagrange es un método para encontrar un polinomio que pase exactamente a través de n puntos dados. Este polinomio es único y tiene grado como máximo n-1.

Ventajas:

  • Conceptualmente simple
  • No requiere resolver sistemas lineales
  • Funciona con puntos no equiespaciados

Desventajas:

  • Computacionalmente más costoso para muchos puntos
  • Puede oscilar mucho (fenómeno de Runge) con muchos puntos

Fórmula de Lagrange

Polinomio interpolante:

ode class="text-whitete fs-6"> P(x) = Σi=0n yi · Li(x)

Donde Li(x) son los polinomios de Lagrange:

ode class="text-white fs-s-6"> Li(x) = Πj=0, j≠in (x - xj) / (xi - xj)

Propiedades:

  • Li(xi) = 1
  • Li(xj) = 0 para i ≠ j
  • Grado del polinomio = n - 1 (donde n es cantidad de puntos)

Ejemplos

Ejemplo 1: Polinomio de Grado 2 (parábola)
  • X: 0 1 2
  • Y: 1 2 4
  • Resultado: P(x) = x² + 1
Ejemplo 2: Polinomio de Grado 3
  • X: 0 1 2 3
  • Y: 1 2 4 7
  • Resultado: Polinomio cúbico interpolante
Ejemplo 3: Datos Sinusoidales
  • X: 0 1.57 3.14 4.71
  • Y: 0 1 0 -1
  • Resultado: Aproximación de sin(x)

Comparación de Métodos de Interpolación

Método Ventajas Desventajas
Lagrange Simple, directo Oscilación, lento para muchos puntos
Newton Rápido, fácil agregar puntos Más complejo de entender
Splines Suave, sin oscilación extrema Requiere resolver sistema lineal

Fenómeno de Runge

Cuando se interpolan muchos puntos (más de 8-10) con Lagrange, el polinomio puede oscilar violentamente cerca de los extremos.

Solución: Usar splines en lugar de un polinomio único.
