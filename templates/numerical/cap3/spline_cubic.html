Descripción

Un Spline Cúbico es una función suave por tramos, donde cada tramo es un polinomio de grado 3 (cúbico). Es el método de interpolación más usado en la práctica porque combina suavidad con estabilidad.

Características:

  • Suave (C² continua) - Primera y segunda derivadas continuas
  • Natural - Condiciones de frontera que evitan oscilación
  • Estable - No sufre del fenómeno de Runge
  • Preciso - Buena aproximación de funciones suaves
  • Más complejo - Requiere resolver un sistema lineal

Fórmula del Spline Cúbico

Para cada intervalo [xi, xi+1]:

ode class="text-white fsfs-6"> Si(x) = ai + bi·(x - xi) + ci·(x - xi)² + di·(x - xi)³

Condiciones de continuidad:

  • C⁰: Si(xi+1) = Si+1(xi+1) (puntos coinciden)
  • C¹: S'i(xi+1) = S'i+1(xi+1) (primera derivada continua)
  • C²: S''i(xi+1) = S''i+1(xi+1) (segunda derivada continua)

Condiciones de frontera "Natural":

  • S''(x₀) = 0 (curvatura cero al inicio)
  • S''(xn) = 0 (curvatura cero al final)

Ejemplos

Ejemplo 1: Función Suave Simple
  • X: 0 1 2 3
  • Y: 0 1 2 3
  • Resultado: Aproximadamente lineal (pendiente constante)
Ejemplo 2: Función Curva
  • X: 0 1 2 3 4
  • Y: 0 1 4 2 5
  • Resultado: Curva suave que pasa por todos los puntos
Ejemplo 3: Función Sinusoidal
  • X: 0 0.5 1 1.5 2
  • Y: 0 0.5 1 0.5 0
  • Resultado: Aproximación suave de media onda sinusoidal

Aspecto Lagrange Spline Lineal Spline Cúbico
Grado del Polinomio n-1 (global) 1 (por tramo) 3 (por tramo)
Continuidad C∞ (suave) C⁰ (puntos) C² (muy suave)
Oscilación Alta para muchos puntos Ninguna Mínima
Complejidad Baja Muy baja Media
Uso Típico Pocos puntos Datos discretos CAD, Gráficos, Ciencia

¿Por qué el Spline Cúbico?

  • Grado 3 es el mínimo para garantizar C² continuidad (derivadas suaves)
  • Grado 4+ sería excesivo (complejidad sin beneficio)
  • Usado industrialmente en CAD, diseño gráfico, física computacional
  • No sufre oscilación como Lagrange con muchos puntos
